# LiveChat

#### Real-time chat application developed using Firebase, Swift and SwiftUI.
MVVM Architecture, IOS 15.0, device: iphone.


## Includes ( ALL SWIFTUI )
- Combine Framework
- Firebase Auth + Database + Storage
- Custom Modifiers
- Animations
- Image picker 
- Light and Dark Mode
